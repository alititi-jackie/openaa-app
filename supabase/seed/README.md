# Supabase Seed Files

These files are safe baseline seeds for local development and review. They do not contain real users, real keys, old Supabase data, or imported legacy content.

The migration files `011` to `013` contain the same baseline concepts for first installation:

- default city
- public site settings
- feature flags
- admin permission keys
- role permission defaults

DMV real question data is not included in Phase 2. The audited legacy JSON is migration input only and will be converted in a later import step.
